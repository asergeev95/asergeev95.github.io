Обозначим через m&n по­раз­ряд­ную конъ­юнк­цию не­от­ри­ца­тель­ных целых чисел m и n. Например, 14&5 = 11102&01012 = 01002 = 4. Для ка­ко­го наи­мень­ше­го не­от­ри­ца­тель­но­го це­ло­го числа А формула
x&9 = 0 → (x&19 ≠ 0 → x&А ≠ 0)
тождественно ис­тин­на (т. е. при­ни­ма­ет зна­че­ние 1 при любом не­от­ри­ца­тель­ном целом зна­че­нии пе­ре­мен­ной х)?
Пусть A: x&A = 0; B: x&9 = 0; C: x&19 = 0.
Имеем: B → (¬C → ¬A).
Упростим выражение: B → (¬C → ¬A) = B → (C ∨ ¬A) = ¬B ∨ C ∨ ¬A.
Рассмотрим уравнение, со­от­вет­ству­ю­щее вы­ра­же­нию ¬B: x&910 ≠ 0 ⇔ x&10012 ≠ 0. Заметим, что это урав­не­ние не вы­пол­ня­ет­ся при x вида x0xx0.
Рассмотрим уравнение, со­от­вет­ству­ю­щее вы­ра­же­нию C: x&1910 = 0 ⇔ x&100112 = 0. Заметим, что это урав­не­ние не вы­пол­ня­ет­ся при x вида 1xxxx, xxx1x, xxxx1.
Значит, вы­ра­же­ние ¬B ∨ C не вы­пол­ня­ет­ся для x вида 10xx0 и x0x10. Для x та­ко­го вида вы­ра­же­ние ¬A долж­но быть равно 1, то есть долж­но вы­пол­нять­ся урав­не­ние x&А ≠ 0. Чтобы это урав­не­ние вы­пол­ня­лось для ука­зан­ных x во вто­ром и пятом раз­ря­дах справа долж­ны сто­ять единицы.
Таким образом, 100102 = 1810 и будет яв­лять­ся ми­ни­маль­ным A при ко­то­ром вы­пол­ня­ет­ся усло­вие задания.
