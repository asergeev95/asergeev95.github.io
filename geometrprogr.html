Определите, что будет напечатано в результате работы следующего фрагмента программы:

 

var k, s: integer;

begin

    s:=0;

    k:=0;

    while s < 80 do begin

        s:=s+2*k;

        k:=k+4;

    end;

    write(s);

end.

Цикл while выполняется до тех пор, пока истинно условие s < 80, т. е. переменная k определяет, сколько раз выполнится цикл. Аккуратно выпишем все s и k. Oтвет 80
Определите, что будет на­пе­ча­та­но в ре­зуль­та­те вы­пол­не­ния программы 
#include <stdio.h>
void main()
{
    int n, s;
    n = 1;
    s = 0;
    while (n <= 100)
    {
        s = s + 30;
        n = n * 2;
    }
    printf("%d", s);
}
Цикл while вы­пол­ня­ет­ся до тех пор, пока ис­тин­но усло­вие n ≤ 100, т. е. пе­ре­мен­ная n определяет, сколь­ко раз вы­пол­нит­ся цикл. По­сколь­ку из­на­чаль­но n = 1, цикл вы­пол­нит­ся 7 раз, следовательно, s = 30 · 7 = 210. Ответ: 210.
